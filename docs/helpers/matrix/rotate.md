__Notes__

+ `rotate` transformations
    * `rotate: a` === `rotateZ`
    * `rotateX: a`
    * `rotateY: a`
    * `rotateZ: a`
    * `rotate3d: x y z a`
+ Only accepts `deg` or `rad` values
