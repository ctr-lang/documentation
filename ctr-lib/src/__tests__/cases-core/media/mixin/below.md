__Description__: The `below: <value>` mixin should create a `(max-width: <value>)` media query

Notation
Object
```
ctr({
  media: {
    query: {
      mixin: {
        below: <value>
      }
    }
  }
})
```
Shorthand
```
ctr({
  media: {
    query: {
      below: <value>
    }
  }
})
```
