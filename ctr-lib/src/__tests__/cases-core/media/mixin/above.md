__Description__: The `above: <value>` mixin should create a `(min-width: <value>)` media query

Notation
Object
```
ctr({
  media: {
    query: {
      mixin: {
        above: <value>
      }
    }
  }
})
```
Shorthand
```
ctr({
  media: {
    query: {
      above: <value>
    }
  }
})
```
