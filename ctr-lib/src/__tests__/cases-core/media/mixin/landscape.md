__Description__: The `landscape: true` mixin should create a orientation media query as such `(orientation: landscape)`

Notation
Object
```
ctr({
  media: {
    query: {
      mixin: {
        landscape: true
      }
    }
  }
})
```
Shorthand
```
ctr({
  media: {
    query: {
      landscape: true
    }
  }
})
```
